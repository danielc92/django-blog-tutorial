# Setup
**How to obtain this repository:**
```sh
git clone https//link.to.this.projects.git-repo
```
**Modules/dependencies:**
- `django`
- `mysqlclient`

**Activate virtualenv and install dependencies**
```sh
cd clone/path
source venv/bin/activate
pip install django mysqlclient
```
